# Baixar Vídeos YT

Um aplicativo simples com interface gráfica para baixar vídeos do YouTube na maior resolução disponível.

## Pré-requisitos

Instale as dependências:
```bash
pip install FreeSimpleGUI yt-dlp
```

## Como usar

Execute o script:
```bash
python Codigo.py
```
Insira a URL do vídeo do YouTube e clique em **Download**. O vídeo será salvo na pasta `Videos` do seu usuário.
